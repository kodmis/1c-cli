Справочник содержит информацию о дополнительных характеристиках элементов номенклатуры и предназначен для демонстрации использования подсистемы Загрузка данных из файла.

В подсистеме Загрузка данных из файла справочник показывает пример загрузки характеристик номенклатуры при загрузке данных в справочник Демо: Номенклатура.

Поле характеристики становится доступным для загрузки при включенной функциональной опции Использовать характеристики. Для каждой позиции номенклатуры доступны для загрузки только свои варианты характеристик ранее определенные в этом справочнике.
