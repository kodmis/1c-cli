Справочник предназначен для демонстрации примера использования ряда подсистем.

В подсистеме Свойства справочник демонстрирует пример объекта, определяющего набор дополнительных реквизитов и сведений (реквизит НаборСвойств) для другого объекта - справочника Демо: Номенклатура.

В подсистеме Управление доступом элементы справочника являются значениями доступа.

  • Значения доступа Демо: Виды номенклатуры назначаются в группах доступа в списке разрешенных значений по виду доступа Демо: Группы номенклатуры с помощью элементов Демо: Группы доступа номенклатуры.
  • Ограничение по виду доступа Демо: Группы номенклатуры используется в списках Демо: Номенклатура, Демо: Виды номенклатуры и Присоединенные файлы (Демо: Номенклатура).
  • Чтение списка Демо: Виды номенклатуры не ограничивается.
  • Изменение списка Демо: Виды номенклатуры ограничивается по одному виду доступа:
    • Демо: Группы номенклатуры (проверяется значение реквизита Ссылка).
