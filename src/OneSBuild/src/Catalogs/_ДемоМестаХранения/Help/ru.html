Справочник предназначен для примера использования ряда подсистем.

В подсистеме Даты запрета изменения элементы справочника используются в качестве "разреза" дат запрета изменения Складской учет, для документов Демо: Оприходование товаров, Демо: Перемещение товаров, Демо: Списание товаров и записей регистра Демо: Заведующие местами хранения.

В подсистеме Обмен данными элементы справочника используются в качестве объектов, синхронизируемых с другими информационными базами.

В подсистеме Префиксация объектов к коду элементов справочника добавляется префикс информационной базы.

В подсистеме Управление доступом элементы справочника выступают в качестве значений доступа.

  • Значения доступа Демо: Места хранения непосредственно назначаются в группах доступа в списке разрешенных значений по виду доступа Демо: Места хранения.
  • Ограничение по виду доступа Демо: Места хранения используется в списках Демо: Места хранения, Демо: Поступление товаров, Демо: Реализация товаров, Демо: Оприходование товаров, Демо: Перемещение товаров, Демо: Списание товаров, Демо: Остатки товаров в местах хранения.
  • Список ограничивается по одному виду доступа:
    • Демо: Места хранения (проверяется значение реквизита Ссылка).
