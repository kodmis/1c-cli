Справочник предназначен для демонстрации примера использования ряда подсистем.

В подсистеме Работа с файлами справочник используется для хранения вложений к документам Демо: Счет на оплату покупателю.

В подсистеме Управление доступом справочник показывает пример ограничения присоединенных файлов, для которых логика ограничения совпадает с ограничением объекта-владельца вложений. В том числе, показывает пример различий в ограничениях для сеансов разного назначения (для пользователей и для внешних пользователей).

  • Для пользователей список ограничивается по двум видам доступа:
    • Организации (проверяется значение реквизита ВладелецФайла.Организация);
    • Демо: Группы партнеров (проверяется значение реквизита ВладелецФайла.Партнер);
    • все значения должны быть разрешенными одновременно (в одной из групп доступа).
  • Для внешних пользователей список ограничивается по одному виду доступа:
    • Внешние пользователи (проверяется значение реквизита Автор).
