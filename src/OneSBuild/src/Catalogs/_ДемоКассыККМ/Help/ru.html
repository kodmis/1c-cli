Справочник предназначен для примера использования ряда подсистем.

В подсистеме Запрет редактирование реквизитов объектов справочник показывает пример того, как блокируется изменение реквизита Принимать выручку общей суммой, который представлен в форме элемента переключателем Розничная выручка из кассы ККМ передается в кассу предприятия, напрямую не связанным с реквизитом.

В подсистеме Управление доступом элементы справочника являются значениями доступа.

  • Значения доступа Демо: Кассы ККМ непосредственно назначаются в группах доступа в списке разрешенных значений по виду доступа Демо: Кассы.
  • Ограничение по виду доступа Демо: Кассы используется в списке Демо: Расходный кассовый ордер.
  • Список Демо: Кассы ККМ ограничивается по одному виду доступа:
    • Организации (проверяется значение реквизита Владелец).
