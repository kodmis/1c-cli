Справочник предназначен для примера использования ряда подсистем.

В подсистеме Даты запрета изменения элементы справочника используются как "разрез" дат запрета изменения Торговля, который используется в документах Демо: Заказ покупателя, Демо: Поступление товаров, Демо: Реализация товаров.

В подсистеме Управление доступом справочник используется в качестве списка групп значений доступа, используемых для снижения трудоемкости указания разрешенных значений по виду доступа Демо: Группа партнеров в группах доступа, вместо партнеров, которых может быть очень много(использование вида доступа см. в справочнике Демо: Партнеры).

  • Значения доступа - это партнеры, а группы (значений) доступа - элементы справочника Демо: Группы доступа партнеров (не путать со справочником Группы доступа, который имеет другой смысл).
  • При редактировании списка разрешенных значений доступа в справочнике Группы доступа элементы справочника Демо: Группа доступа партнеров используется вместо значений доступа Демо: Партнеры.
  • Список Демо: Группа доступа партнеров ограничивается по одному виду доступа:
    • Демо: Группы партнеров (проверяется значение реквизита Ссылка);
    • при этом ограничиваются только элементы справочника, а группы элементов (папки) не ограничиваются.
