Справочник предназначен для демонстрации примера использования ряда подсистем.

В подсистеме Работа с файлами справочник используется для хранения вложений к элементам справочника Демо: Номенклатура, а также демонстрируют процедуру перехода от хранения файлов в общем справочнике Файлы в этот справочник. Такой переход может понадобиться при пересмотре варианта хранения файлов, например, при усложнении логики RLS. По команде Перенести файлы (обработка Демо: Перенос файлов) можно перенести ранее созданные элементы справочника Файлы в справочник Присоединенные файлы (Демо: Номенклатура) . Такая операция также может быть осуществлена автоматически, например, с помощью отложенного обработчика обновления.

В подсистеме Управление доступом справочник показывает пример ограничения присоединенных файлов, для которых логика ограничения совпадает с ограничением объекта-владельца вложений.

  • Список ограничивается по одному виду доступа:
    • Демо: Группы номенклатуры (проверяется значение реквизита ВладелецФайла).
