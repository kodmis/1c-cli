Справочник предназначен для примера использования подсистемы Управление доступом.

В подсистеме Управление доступом справочник используется в качестве списка групп значений доступа, используемых для снижения трудоемкости указания разрешенных значений по виду доступа Демо: Группы номенклатуры в группах доступа, вместо номенклатуры, которой может быть очень много (использование вида доступа см. в справочниках Демо: Номенклатура и Демо: Виды номенклатуры).

  • Значения доступа - это номенклатура и виды номенклатуры, а группы (значений) доступа - элементы справочника Демо: Группы доступа номенклатуры (не путать со справочником Группы доступа, который имеет другой смысл).
  • При редактировании списка разрешенных значений доступа в справочнике Группы доступа элементы справочника Демо: Группа доступа номенклатуры используются вместо значений доступа Демо: Номенклатура и Демо: Виды номенклатуры.
  • В данном примере один справочник Демо: Группа доступа номенклатуры используется для двух типов значений доступа Демо: Номенклатура и Демо: Виды номенклатуры.
  • Список Демо: Группа доступа номенклатуры не ограничивается на уровне записей.
