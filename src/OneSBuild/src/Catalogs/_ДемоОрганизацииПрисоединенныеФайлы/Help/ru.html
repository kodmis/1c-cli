Справочник предназначен для демонстрации примера использования ряда подсистем.

В подсистеме Работа с файлами справочник используется для хранения вложений к элементам справочника Демо: Проекты. Для комфортабельного хранения большого количества файлов предусмотрена возможность создавать папки.

В подсистеме Управление доступом справочник показывает пример ограничения присоединенных файлов, для которых логика ограничения устанавливается по объекту-владельцу с зависимостью права Изменение присоединенного файла от права Чтение объекта-владельца.

  • Список ограничивается по специальному виду доступа Объект:
    • Проверяется значение реквизита ВладелецФайла.
    • В процедуре ПриЗаполненииЗависимостейПравДоступа общего модуля УправлениеДоступомПереопределяемый указано, что право Изменение присоединенного файла зависит от права Чтение объекта-владельца.
    • В процедуре ПриЗаполненииОграниченияДоступа модуля менеджера объекта-владельца записываются наборы значений доступа для работы специального вида доступа Объект в ограничении присоединенных файлов.
