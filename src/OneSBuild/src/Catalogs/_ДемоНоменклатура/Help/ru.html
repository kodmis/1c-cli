Справочник предназначен для демонстрации примера использования ряда подсистем.

В подсистеме Групповое изменение объектов элементы справочника демонстрируют возможность группового изменения любых реквизитов, кроме реквизита СкрытыйРеквизит (это исключение задается в модуле менеджера объекта).

В подсистеме Загрузка данных из файла справочник показывает пример загрузки списка аналогов номенклатуры из файлов в табличную часть Аналоги, где состав полей бланка для загрузки динамически определяется в зависимости от свойств номенклатуры.

В подсистеме Запрет редактирования реквизитов объектов справочник показывает пример того, как реквизиты ВидНоменклатуры и Код могут быть заблокированы для изменения.

В подсистеме Контактная информация форма элемента справочника демонстрирует выбор страны происхождения товара из справочника Страны мира, а также автоматическое создание его элементов по данным классификатора ОКСМ при выборе страны происхождения товара с помощью автоподбора.

В подсистеме Обмен данными демонстрирует возможности обмена совместно разделенных данных.

В подсистеме Оценка производительности замеряется время открытия формы элемента (ключевая операция Демо: открытие фирмы элемента).

Подсистема Поиск и удаление дублей демонстрирует возможность объединения дублей, замены элементов и поиск мест использования прямо из списка номенклатуры. В качестве примера предусмотрен запрет на замену номенклатуры с кодом 000000001 (это исключение задается в модуле менеджера объекта). Также в модуле менеджера объекта проверяется, чтобы вид номенклатуры у объединяемых элементов был одинаков, и дан пример определения прикладных параметров поиска дублей.

В подсистеме Префиксация объектов к номеру справочника добавляется префикс информационной базы.

В подсистеме Работа с файлами элементы справочника используются в качестве объектов-владельцев файлов-вложений расположенных в справочнике Присоединенные файлы (Демо: Номенклатура), а также демонстрируют процедуру перехода от хранения файлов в общем справочнике Файлы в отдельные справочники. Такой переход может понадобиться при пересмотре варианта хранения файлов, например, при усложнении логики RLS. По команде Перенести файлы (обработка Демо: Перенос файлов) можно перенести ранее созданные элементы справочника Файлы в справочник Присоединенные файлы (Демо: Номенклатура). Такая операция также может быть осуществлена автоматически, например, с помощью отложенного обработчика обновления. Реквизит ФайлКартинки демонстрирует возможность хранения изображения товара, в справочнике Присоединенные файлы (Демо: Номенклатура), для его последующего вывода на карточке товара.

Кроме того, в форме элемента приведены примеры отображения гиперссылки присоединенных файлов и картинки из присоединенного файла.

В подсистеме Свойства элементы справочника являются объектами-владельцами дополнительных реквизитов и сведений.
См. также вариант отложенной инициализации дополнительных реквизитов в справочнике Демо: Контрагенты.

В подсистеме Склонение представлений объектов элементы справочника демонстрируют создание, редактирование и сохранение представления элемента во всех падежах русского языка.

В подсистеме Удаление помеченных объектов элементы справочника демонстрируют удаление циклических ссылок из табличной части Аналоги.

Кроме того в форме списка приведен пример встраивания механизма скрытия помеченных на удаление объектов с использованием подсистемы Подключаемые команды.

В подсистеме Управление доступом элементы справочника являются значениями доступа.

  • Значения доступа Демо: Номенклатура назначаются в группах доступа в списке разрешенных значений по виду доступа Демо: Группы номенклатуры с помощью элементов Демо: Группы доступа номенклатуры.
  • Ограничение по виду доступа Демо: Группы номенклатуры используется в списках Демо: Номенклатура, Демо: Виды номенклатуры и Присоединенные файлы (Демо: Номенклатура).
  • Чтение списка Демо: Номенклатура не ограничивается.
  • Изменение списка Демо: Номенклатура ограничивается по одному виду доступа:
    • Демо: Группы номенклатуры (проверяется значение реквизита Ссылка).
