Обработка предназначена для примера использования подсистемы Печать (демонстрация обратной совместимости).

С ее помощью добавляется команда Печать: Списание товаров (с помощью обработки _ДемоПечатнаяФорма) в документ _ДемоСписаниеТоваров. Команда добавляется с помощью устаревшего способа внедрения - через команду Обработка._ДемоПечатнаяФорма.СписаниеТоваровMSWord. Для штатного способа внедрения см. документ Демо: Счет на оплату покупателю.
