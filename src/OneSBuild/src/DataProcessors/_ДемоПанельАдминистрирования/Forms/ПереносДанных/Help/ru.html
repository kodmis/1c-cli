В данной панели осуществляется перенос данных в другие программы, переход с других программ.

  • Демо: Перенос сведений о пользователях в другую программу
    		•  Перенос сведений о пользователях в другую программу "Демонстрационная программа "Библиотека стандартных подсистем" с использованием обработки УниверсальныйОбменДаннымиXML.

    Загрузка данных

  • Загрузка данных из файлов в справочники
    		•

    Загрузка табличных данных в произвольные справочники из файлов (Microsoft Office Excel, Microsoft Office Word, OpenOffice Calc и др.). Ввод данных возможен двумя способами:
    1) Копирование данных в таблицу из внешнего файла через буфер обмена (используя команды "Копировать"и "Вставить").
    2) Сохранить бланк в файл для заполнения в другой программе. После редактирования загрузить готовые данные из файла в таблицу.
