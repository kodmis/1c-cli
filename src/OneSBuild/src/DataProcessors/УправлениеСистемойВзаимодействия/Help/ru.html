Обработка предназначена:
- для подключения / отключения информационной базы (приложения) к системе взаимодействия;
- для отмены регистрации информационной базы (приложения) от системы взаимодействия;
- для настройки совместного использования приложений одного абонента;
- для просмотра списка пользователей и блокировки/разблокировки пользователей;
- для сопоставления пользователей локальной базы и пользователей сервиса Фреш

Подключение информационной базы

В обработке поддерживается 3 способа регистрации информационной базы:

1 способ - при помощи электронной почты и кода подтверждения. В этом случае в обработке нужно заполнить адрес электронной почты абонента, указать наименование приложения, нажать кнопку Получить код. На указанный адрес электронной почты будет выслан код регистрации. Полученный код нужно ввести в поле Код регистрации и нажать кнопку Зарегистрировать.
При указанном способе регистрации база будет привязана к указанному адресу электронной почты, совместное использование локальной базы и приложений в Фреш при таком способе регистрации будет невозможным.

2 способ - через сервис Фреш (в локальной базе). Этот способ доступен для абонентов Фреш, (в том случае, если в сервисе Фреш поддерживается данная возможность). Для регистрации необходим код, который можно получить в менеджере сервиса (для этого в менеджере сервиса в обработке "Управление системой взаимодействия" нужно нажать кнопку Получить код для локальной информационной базы, скопировать полученный код в буфер обмена и вставить его в поле Код управления в данной обработке. После ввода кода управления должно автоматически заполниться поле Адрес менеджера сервиса. После этого нужно нажать кнопку Зарегистрировать.

3 способ - через сервис Фреш (в приложении Фреш). Для регистрации достаточно нажать кнопку Зарегистрировать.

Регистрация через сервис Фреш позволит совместно использовать систему взаимодействия в приложениях в сервисе и в локальных базах.

Отмена регистрации

Для отмены регистрации достаточно нажать гиперссылку Отменить регистрацию и подтвердить действие.

Настройка совместного использования приложений

Система взаимодействия позволяет объединить несколько приложений, зарегистрированных в системе взаимодействия, в единое пространстве. В случае объединения приложений, в качестве пользователей системы взаимодействия выступают пользователи всех объединенных приложений. При объединении приложений, также, становится доступной возможность обмена сообщениями между приложениями. Объединять приложения может только владелец абонента всех объединяемых приложений.
Для настройки совместного использования нужно перейти по гиперссылке Совместное использование приложений. Затем в появившейся таблице нажать кнопку Добавить, отметить приложения, которые нужно объединить для совместного использования. Если нужно настроить сопоставление с пользователями сервиса Фреш, в поле Сопоставление пользователей нужно выбрать значение По ключу сопоставления. После сохранения настроек станет возможным совместное использование указанных приложений.

Просмотр списка пользователей

Для просмотра списка пользователей системы взаимодействия можно перейти по гиперссылке Пользователи. В открывшейся таблице отображаются пользователи информационной базы (приложения) и пользователи системы взаимодействия. Отображаются следующие данные: имя и полное имя пользователя; признак того, что пользователь является пользователем информационной базы и системы взаимодействия, а также признак того, что пользователь заблокирован. Есть возможность заблокировать / отменить блокировку пользователей. Для этого используются соответствующие кнопки над таблицей.

Сопоставление с пользователями Фреш

Такое сопоставление может быть полезным, если нужно организовать переписку между пользователями локальной базы и пользователями приложений Фреш. При этом необходимо, чтобы было настроено совместное использование приложений (см. п. Настройка совместного использования приложений).
Для сопоставления пользователей нужно ввести код управления (может быть получен в менеджере сервиса таким же способом, как и при регистрации локальной базы через сервис Фреш).
После этого нажать на кнопку Получить информацию о пользователях сервиса. После этого будет заполнен список пользователей локальной базы. В колонке "Пользователь сервиса" можно выбрать соответствующего пользователя сервиса.
После настройки соответствия, его необходимо сохранить (при помощи кнопки Сохранить)
После этого пользователи локальной базы будут соответствовать указанным пользователь сервиса. Обсуждения локального пользователя будут доступны соответствующему пользователю Фреш.
