Очистка списка версий позволяет имитировать "пустую" базу. При перезапуске будет выполнена процедура первоначального заполнения ИБ. Уменьшение номера версии ИБ позволяет выполнить процедуру обновления с этого номера версии до текущей версии конфигурации.

Все настроечные флажки запоминаются и действуют только для текущего пользователя.
