Обработка демонстрирует процедуру перехода от хранения файлов в общем справочнике Файлы в отдельные справочники. Такой переход может понадобиться при пересмотре варианта хранения файлов, например, при усложнении логики RLS. По команде Перенести файлы можно перенести ранее созданные файлы справочника Файлы в справочник Присоединенные файлы (Демо: Номенклатура). Такая операция также может быть осуществлена автоматически, например, с помощью отложенного обработчика обновления.

Пример использования.

  1. Выбрать в поле Объект метаданных с файлами справочник Демо: Номенклатура.
  2. В списке будут выведены все элементы справочника Демо:Номенклатура, к которым присоединены элементы справочника Файлы.
  3. По кнопке Перенести файлы все файлы в указанных элементах будут перенесены в справочник Присоединенные файлы (Демо: Номенклатура), а их оригиналы будут помечены на удаление.
