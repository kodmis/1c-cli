Обработка подключается к конфигурации средствами подсистемы Дополнительные отчеты и обработки (вид: ДополнительнаяОбработка) и поставляет следующие команды:

Команда Загрузить номенклатуру из прайс-листа фирмы "1C" (профили безопасности) (тип: ВызовСерверногоМетода) демонстрирует выполнение команд в безопасном режиме с использованием дополнительных разрешений для получения данных из интернета и чтения Excel в глобальном серверном контексте в фоновом задании.

Команда Параметры загрузки номенклатуры из прайс-листа (профили безопасности) (тип: ОткрытиеФормы) демонстрирует открытие основной формы внешнего объекта в глобальном контексте для сохранения и ввода параметров, а также выполнение серверной команды внешнего объекта из его формы.

Для использования на сервере должен быть установлен Microsoft Office или Data Connectivity Components.
