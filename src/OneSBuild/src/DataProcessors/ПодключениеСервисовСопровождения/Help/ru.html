Предназначена для подключения тестовых периодов сервисов сопровождения.

Подключения тестового периода отображается для не подключенных сервисов, если есть возможность активации тестового периода. Обработка позволяет выбрать тестовый период из списка доступных пользователю и выполнить подключение.
