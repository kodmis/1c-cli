Обработка подключается к конфигурации средствами подсистемы Дополнительные отчеты и обработки (вид: ЗаполнениеОбъекта) и поставляет следующие команды для заполнения элементов справочника Демо: Контрагенты:

Команда Заполнить реквизит "Полное наименование" (тип: ВызовСерверногоМетода) демонстрирует заполнение реквизитов одного или нескольких объектов по ссылкам в серверном контексте с отображением всплывающего оповещения.

Команда Добавить префикс к реквизиту "Наименование" (тип: ОткрытиеФормы) демонстрирует открытие основной формы внешнего объекта для заполнения реквизитов одного или нескольких объектов по ссылкам.

Команда Комплексная очистка (тип: ВызовСерверногоМетода) демонстрирует заполнение реквизитов одного или нескольких объектов по ссылкам в серверном контексте без отображения всплывающего оповещения.

Команда Комплексное заполнение (тип: ВызовКлиентскогоМетода) демонстрирует заполнение реквизитов одного или нескольких объектов по ссылкам в клиентском контексте с отображением всплывающего оповещения.

Команда Заполнить реквизит "ИНН" не записывая объект (тип: ЗаполнениеФормы) демонстрирует заполнение реквизитов одного объекта из его карточки в серверном контексте без отображения всплывающего оповещения.
