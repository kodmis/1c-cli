В подсистеме Варианты отчетов на базе отчета демонстрируются возможности подмены схемы отчета и настроек основного варианта.

Для формирования отчета нажмите кнопку Сформировать.

При необходимости установите параметры отчета:

Автор Будут отобраны только файлы по указанному автору
Автор версии Будут отобраны только такие файлы, автором текущей версии которых является указанный пользователь
Редактирует Будут отобраны файлы, занятые для редактирования указанным пользователем
Период Будут отобраны только такие файлы, созданные за указанный период

 

Для детальной настройки отчета используйте команду Еще - Изменить вариант.
