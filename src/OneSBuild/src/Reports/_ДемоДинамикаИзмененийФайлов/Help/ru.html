В подсистеме Варианты отчетов на базе отчета демонстрируются возможности по ограничению списка значений доступных, для выбора, работа с диаграммами, а также фоновое формирование "тяжелых" отчетов (более 4 секунд).

Для формирования отчета нажмите кнопку Сформировать. По умолчанию отчет показывает динамику роста хранилища файлов в течение текущего месяца.

Для детальной настройки отчета используйте команду Еще - Изменить вариант.
