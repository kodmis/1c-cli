Предназначен для демонстрации использования ряда подсистем.

В подсистеме Обмен данными демонстрируется пример регистра бухгалтерии, данные которого синхронизируются с другими информационными базами.

Подсистема Поиск и удаление дублей демонстрирует на примере этого регистра результат объединения дублей в плане счетов, организациях и валютах. Также проанализировать данные бухгалтерского учета можно с помощью отчета Демо: Оборотно-сальдовая ведомость.
