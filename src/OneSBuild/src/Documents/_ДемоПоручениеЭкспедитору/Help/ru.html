Документ предназначен для примера использования подсистемы Управление доступом.

Документ показывает пример реализации ограничения доступа по полю Пункт составного с использованием типа Строка.
Список документов ограничивается по двум видам доступа:

    • Демо: Места хранения (проверяются значения реквизитов Место хранения и Пункт);
    • Демо: Группы партнеров (проверяется значение реквизита Пункт).
    • Демо: Физические лица (проверяется значения реквизита Пункт).
    • Демо: Подразделения (проверяется значения реквизита Пункт).
    • все значения должны быть разрешенными одновременно (в одной из групп доступа).
