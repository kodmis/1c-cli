Документ предназначен для примера использования подсистемы Даты запрета изменения, Управление доступом.

В подсистеме Даты запрета изменения документ показывает пример запрета изменения документа по разделу дат запрета Демо: Начисление зарплаты для случая, когда раздел не содержит уточнения по объектам.

В подсистеме Управление доступом документ показывает пример реализации ограничения доступа с использованием табличной части. Кроме того, проверка ограничения сделана с помощью универсального шаблона #ПоНаборамЗначений, для чего создана еще одна специальная табличная часть НаборыЗначенийДоступа.

  • Список ограничивается по двум видам доступа:
    • Пользователи (проверяется значение реквизита Ответственный);
    • Демо: Физические лица (проверяются значения реквизита Физическое лицо табличной части Зарплата).
    • все значения должны быть разрешенными одновременно (в одной из групп доступа).
