Документ предназначен для примера использования подсистемы Даты запрета изменения.

В подсистеме Даты запрета изменения документ показывает пример условного отказа от запрета изменения документа в зависимости от флажка Проведено банком, при условии, что в качестве даты, используемой при проверке применяется Дата проведения банком, которая не заполняется при сброшенном флажке Проведено банком. Запрет проверяется по разделу дат запрета Демо: Банк по значению реквизита Банковский счет (Демо: Банковский счет).
