Документ предназначен для примера использования ряда подсистем.

В подсистеме Управление доступом документ показывает пример реализации ограничения по одному виду доступа, когда проверяемое значение находится за пределами документа.

  • Список ограничивается по одному виду доступа:
    • Демо: Группы партнеров (проверяется значение реквизита Продавец.Партнер).
