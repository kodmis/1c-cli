Регистр накопления предназначен для примера использования ряда подсистем.

В подсистеме Обмен данными демонстрируется пример регистра накопления, данные которого синхронизируются с другими информационными базами.

В подсистеме Управление итогами и агрегатами для этого регистра накопления определены агрегаты, которые возможно централизованно обновлять и настраивать.
