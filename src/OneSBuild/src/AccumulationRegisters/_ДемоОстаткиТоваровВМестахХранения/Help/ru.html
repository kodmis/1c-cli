Регистр накопления предназначен для демонстрации использования ряда подсистем.

Подсистема Поиск и удаление дублей демонстрирует на примере этого регистра результат объединения дублей в организациях, местах хранения и номенклатуре.

В подсистеме Управление доступом регистр реализует пример логики ограничения регистров накопления по измерениям с использованием шаблона ограничения доступа #ПоЗначениям.

  • Список ограничивается по двум видам доступа:
    • Организации (проверяется значение реквизита Организация);
    • Демо: Места хранения (проверяется значение реквизита Место хранения);
    • все значения должны быть разрешенными одновременно (в одной из групп доступа).
