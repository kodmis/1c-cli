Для редактирования персональных настроек можно воспользоваться командой Демо: Мои настройки в разделе Главное - группа Сервис панели действий, администратор может найти эту команду в разделе Администрирование - группа Сервис панели действий.

Главное

На вкладке доступны настройки и команды:

  • Учетная запись - с помощью этой ссылки можно ввести в программу данные о себе. Можно просмотреть параметры для входа в программу, сменить пароль (если это разрешено администратором), ввести контактную информацию, просмотретьсостав прав доступа (группу доступа и профиль доступа) и вывести отчет по своим правам доступа. Если с программой работает внешний пользователь, на просмотр открываются данные внешнего пользователя.

Рабочая дата

  • С помощью переключателя Рабочая дата можно поменять дату в программе, которая будет подставляться при создании и копировании документов:
    • Текущая дата компьютера - можно выбрать текущую дату, установленную на компьютере;
    • Другая дата - можно ввести дату вручную или выбрать из календаря с помощью кнопки .
  • Настройка действует для следующих видов документов:
      • Начисление зарплаты;
      • Оприходование товаров;
      • Поступление товаров;
      • Перемещение товаров;
      • Списание товаров.

Запуск и завершение работы

  • С помощью флажков можно определить, как будет происходить Запуск и завершение работы:
    • Демо: Предлагать перейти на сайт при запуске - в этом случае после открытия программы будет выведено сообщение с предложением перейти на сайт программы для ознакомления с ее документацией. При этом работа программы блокируется. По умолчанию флажок выключен.
    • Запрашивать подтверждение при завершении программы - включите флажок, для того чтобы при выходе из программы требовалось подтверждение (рекомендуется). По умолчанию флажок включен. Для предотвращения случайного закрытия главного окна программы, перед завершением работы, в программе предусмотрен запрос подтверждения работы.

Работа в веб-клиенте

  • Для повышения удобства работы в веб-клиенте требуется установка и подключение расширения для веб-клиента. Требуется для выполнения определенных операций с файлами в веб-клиенте.
  • При работе с файлами через веб-клиент необходимо учитывать особенности, которые связаны с моделью безопасности веб-браузера.
  • Например, по умолчанию такие операции, как получение файлов на клиентский компьютер или передача файлов на сервер, могут работать с ограничениями.
  • Для того чтобы снять эти ограничения, можно установить себе на клиентский компьютер специальное расширение для работы с файлами, которое входит в состав дистрибутива платформы 1С:Предприятие 8.3. Предусмотрены команды:
    • Напоминать об установке расширения для веб-клиента 1С:Предприятие - если флажок включен, то программа выводит соответствующее сообщение (по умолчанию флажок включен, если установка расширения еще не производилась). Не рекомендуется выключать флажок, т.к. без данного расширения работа в веб-клиенте невозможна;
    • Установить сейчас - Нажмите кнопку, для того чтобы сразу установить это расширение.
  • После установки расширения флажок и кнопка установки становятся недоступными, выводится информация, что расширение установлено.

Доступ к внешним ресурсам

В разделе сгруппированы команды для настройки доступа к различным внешним ресурсам:

  • Настроить доступ к Интернету - переход к настройке параметров прокси-сервера.
  • Настройки работы с почтой (при наличии прав доступа) - переход к персональным настройкам работы с почтой (включение подписи для новых сообщений, включение подписи при ответе или пересылке, настройки уведомлений для отправляемых сообщений, настройки ответов на запрашиваемые уведомления, включение содержания исходного письма при пересылке).

Прочее

В разделе сгруппированы команды:

  • Настройка электронной подписи и шифрования - переход к настройкам электронной подписи.
  • Настройка 1С-Коннект - переход к персональным настройкам 1С-Коннект.
  • Задать действие при выборе макета печатной формы - указывает, какое действие выполнять при нажатии на клавишу Enter или двойном щелчке мыши на макете печатной формы - открыть макет для просмотра или для просмотра и редактирования.
  • Обновить интерфейс программы - с помощью этой кнопки можно применить ранее измененные для данного пользователя параметры программы. Можно использовать эту команду вместо перезагрузки программы (например, в случае если администратор изменил настройки, и их необходимо применить, не выходя из программы).

Работа с файлами

На вкладке производятся персональные настройки для работы с файлами:
  • Действие при выборе файла - укажите действие программы при выборе файла в списке с помощью нажатия на клавишу Enter или двойном щелчке мыши на файле - открыть файл или открыть его карточку.
  • Выбор режима открытия при выборе файла - включите флажок, для того чтобы программа предлагала выбрать режим открытия файла при его выборе. Если флажок включен, то перед открытием файла откроется окно для выбора режима открытия - просмотр или редактирование.
  • Показывать подсказки при редактировании файлов (только веб-клиент) - включить подсказки при редактировании файлов при работе в веб-клиенте. Если флажок включен, то при работе через веб-клиент программа будет подсказывать, как удобнее открыть файл для просмотра или редактирования, как обновить его в информационной базе.
  • Показывать занятые файлы при завершении работы - если флажок включен и текущий пользователь занял один или несколько файлов, то при завершении работы показывается список этих файлов. Можно открыть файл для просмотра, закончить редактирование, открыть карточку файла.
  • Показывать колонку "Размер" в списках файлов - если флажок включен, в списке файлов и в списке редактируемых файлов показывается колонка "Размер" (в Кб).
  • Показывать информацию, что файл не был изменен - если флажок включен, то при сохранении версий файла программа будет сообщать, что в файл не было внесено изменений. При выполнении команды Закончить редактирование или Сохранить изменения, если файл не был изменен, новая версия не создается (существующая не изменяется), если включен флажок, то выходит сообщение, что файл не менялся.
  • Сравнивать версии файлов ODT, DOC, RTF, TXT при помощи - выбор одного из двух возможных вариантов сравнения файлов указанного формата - Microsoft Office Word или OpenOffice.org Writer.
  • Настройки открытия файлов - в предлагаемой таблице выберите способ открытия для текстовых файлов с указанными расширениями (во встроенном редакторе программы или стандартным приложением, например, текстовый файл вместо встроенного редактора программы можно открыть с помощью Блокнота Windows).
  • Настройка основного рабочего каталога - перейдите по ссылке, для того чтобы указать параметры основного рабочего каталога.
  • Настройка сканирования - переход к настройке параметров сканирования.
