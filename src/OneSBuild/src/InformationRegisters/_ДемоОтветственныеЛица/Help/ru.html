Регистр сведений предназначен для примера использования подсистемы Обмен данными.

В подсистеме Обмен данными регистр сведений демонстрирует пример объекта, который синхронизируется с другими информационными базами.
Если этот регистр не заполнен, то помощник создания обмена данными предлагает его заполнить.
