Регистр сведений предназначен для примера использования ряда подсистем.

В подсистеме Обмен данными регистр показывает пример синхронизации периодического регистра сведений с другими информационными базами.

В подсистеме Управление доступом регистр показывает пример реализации ограничения по двум видам доступа для регистров сведений.

  • Список ограничивается по двум видам доступа:
    • Организации (проверяется значение реквизита Организация);
    • Демо: Физические лица (проверяется значение реквизита Физическое лицо);
    • все значения должны быть разрешенными одновременно (в одной из групп доступа).
