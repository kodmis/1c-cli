Регистр сведений предназначен для примера использования подсистемы Даты запрета изменения.

В подсистеме Даты запрета изменения регистр сведений показывает пример запрета изменения записи по разделу Демо: Складской учет с объектом Демо: Место хранения, который извлекается из измерения Место хранения, а в качестве проверяемой даты используется дата из измерения Период.
