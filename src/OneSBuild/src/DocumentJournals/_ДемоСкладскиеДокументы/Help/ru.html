Журнал документов предназначен для примера использования ряда подсистем.

В подсистеме Печать журнал демонстрирует автоматическое создание подменю Печать на основе видов документов, входящих в состав журнала.

В подсистеме Управление доступом журнал документов показывает пример использования шаблона ограничения доступа #ПоЗначениямИНаборамРасширенный для журналов документов. Для работы шаблона документы включенные в журнал записывают наборы значений доступа. В модулях их объектов встроены процедуры ЗаполнитьНаборыЗначенийДоступа.

  • Список ограничивается по видам доступа, указанных для документов журнала: Демо: Оприходование товаров, Демо: Перемещение товаров, Демо: Списание товаров. При этом ограничение документов в журнале совпадает с ограничением документов в их списках.
